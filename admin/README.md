# 管理员入口使用说明

## 🔐 访问管理后台的步骤

### 1. 进入登录页面
访问 `login.html` 登录页面

### 2. 找到管理员入口
在登录表单底部，会看到一个灰色的链接：**🔐 管理员入口**

### 3. 输入管理员密码
点击"管理员入口"链接后，会弹出密码输入框

**默认管理员密码：** `admin123`

> ⚠️ **安全提示：** 在实际项目中，建议：
> - 将密码存储在后端数据库中
> - 使用加密传输（HTTPS）
> - 实现更复杂的身份验证机制
> - 添加登录失败次数限制

### 4. 验证成功后跳转
密码正确后，系统会：
- 显示"验证成功，跳转到管理后台"提示
- 在 sessionStorage 中保存管理员标记
- 自动跳转到宠物管理页面

### 5. 管理后台功能
成功登录后，可以访问以下管理页面：
- 📋 **宠物管理** (`admin/pet-manage.html`) - 管理宠物信息
- 📝 **申请管理** (`admin/apply-manage.html`) - 审核领养申请
- 👥 **用户管理** (`admin/user-manage.html`) - 管理用户账号
- 💬 **评论管理** (`admin/comment-manage.html`) - 管理评论内容

## 🛡️ 权限保护机制

### SessionStorage 验证
所有管理后台页面都添加了权限验证：
```javascript
const isAdmin = sessionStorage.getItem("isAdmin");
if (!isAdmin || isAdmin !== "true") {
  alert("无权访问，请先通过管理员入口登录！");
  location.href = "../login.html";
  return;
}
```

### 自动重定向
如果未通过管理员入口登录，直接访问管理后台页面会被自动重定向到登录页。

### Session 过期
关闭浏览器后，sessionStorage 会自动清除，需要重新验证身份。

## 🎨 界面特点

- 管理员入口链接采用低调的灰色设计，不会干扰普通用户
- 鼠标悬停时会变为金黄色，提示可点击
- 位于登录表单底部，用分隔线与主要内容区分
- 密码验证使用浏览器原生 prompt，简单直观

## 📝 注意事项

1. **不要将管理员密码泄露给他人**
2. **定期更换管理员密码**
3. **建议在正式环境中使用更安全的认证方式**
4. **sessionStorage 仅在当前标签页有效，关闭即失效**
5. **如需持久化登录，可改用 localStorage（但安全性较低）**

## 🔧 修改管理员密码

如需修改管理员密码，编辑 `login.html` 文件中的这行代码：

```javascript
const correctAdminPassword = "admin123"; // 修改为你的新密码
```

> ⚠️ 实际项目中应该通过后端API进行密码验证，而不是在前端硬编码！
